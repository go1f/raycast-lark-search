# Lark Search Changelog

## [Initial Version] - 2026-06-14

- Search Feishu/Lark docs, messages, chats, and contacts from Raycast.
- Add `/doc`, `/msg`, and `/im` filters for narrowing search scope.
- Add a local Script Commands hot index for high-frequency Lark results in Raycast Root Search.
- Support separate Lark and Feishu desktop app targets with configurable bundle IDs and lark-cli identities.
